# Concepts

_A brief overview of concepts you will need to understand the Velt SDK._

---

### Document

> _A **Document** represents a virtual building. The **Document ID** is like the address of the building that uniquely identifies it._

### Location

> _A **Location** represents a virtual room inside a **Document**. A user can be in multiple **Locations** at once._

### Group

> _A **Group** represents a way to group users together under an identifier, a **Group ID**._

### Plan

> _A **Plan** represents a way to restrict certain features for users based on an identifier, a **Plan ID**._
