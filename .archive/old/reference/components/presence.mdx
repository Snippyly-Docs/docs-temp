# Presence

Description about presence.

## Properties

`containerClass` (deprecated)

`optional`

Used to add custom styles to the presence container.

`maxUsers`

`default: 5`

Used to specify the max users before appending the more users avatar.

`documentParams` (deprecated)

`optional`

Used to set the location that the presence component should filter by.

## Attributes

`dark`

`optional`

Force dark mode on the component.

## Events

`onUsersChanged`

Emits the filtered list of `PresenceUser[]` every time the list changes and is filtered.

## CSS Parts

`snippyly-presence::part(container)`

Targets the root div container.

`snippyly-presence::part(tooltip)`

Targets the container that is hovered over in order to show the tooltip.

`snippyly-presence::part(tooltip-avatar)`

Targets the avatar within the tooltip.

`snippyly-presence::part(user-avatar)`

Targets the user avatar container.

`snippyly-presence::part(more-users)`

Targets the more users container.

## CSS Variables

`--snippyly-presence-avatar-size`

Inherits from: `--snippyly-base-icon-rem`

Default value: `2rem`

`--snippyly-presence-primary-text-color`

`--snippyly-presence-secondary-text-color`
