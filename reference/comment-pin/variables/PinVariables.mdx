| Variable Name | Default | What does it do? |
| ------------- | --------------------- | ---------------- |
| `--snippyly-comment-pin-size` | `2.5rem` | Sets the comment pin size |
| `--snippyly-comment-pin-avatar-size` | `1.5rem` | Sets the comment pin avatar size |
| `--snippyly-comment-pin-bg-color` | `--snippyly-purple` | Set this to `!important` to override status color styling |
| `--snippyly-comment-pin-box-shadow `| `0px 8px 16px -8px rgba(15, 15, 15, 0.1)` | Sets the comment pin box shadow |
| `--snippyly-comment-pin-ghost-color` | `--snippyly-grey-3-color` | Sets the color of ghost comment pins |
| `--snippyly-comment-pin-popover-color` | `#a259fe` | Sets the color of the default popover comment pin triangle |
| `--snippyly-comment-pin-border` | `2px solid white` | Sets the color of the comment pin border |
| `--snippyly-comment-pin-index-border` | `1px solid rgb(210, 210, 210)` | Sets the color of the comment index border |
| `--snippyly-comment-pin-index-font-size` | `0.55rem` | Sets the font size of the comment index |
| `--snippyly-comment-pin-index-bg` | `white` | Sets the background color of the comment index |
| `--snippyly-internal-lock-icon-color` | `white` | Sets the color of the comment pin lock icon |
| `--snippyly-internal-lock-bg-color` | `grey` | Sets the background color of the comment pin lock icon |
